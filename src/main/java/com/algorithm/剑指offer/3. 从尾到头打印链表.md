### Title
输入一个链表的头结点，按照 从尾到头 的顺序返回节点的值。

返回的结果用数组存储。
### Demo
```
输入：[2, 3, 5]
返回：[5, 3, 2]
```
### Analysis
反转链表 -- 基础题
### Code

```java
class Solution {
    public int[] printListReversingly(ListNode head) {
        // 声明三个节点
        ListNode pre = null;
        ListNode node = head;
        ListNode next = null;
        int count = 1;
        while(node.next != null){
            // 先拿出next节点， 然后next节点应该指向pre， pre指向当前节点
            next = node.next;
            node.next = pre;
            pre = node;
            node = next;
            count++;
        }
        node.next = pre;
        int nums [] = new int[count];
        int i = 0;
        while(node != null){
            nums[i++] = node.val;
            node = node.next;
        }
        return nums;
    }
}
```
