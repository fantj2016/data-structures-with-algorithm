### Title
输入一个整数 n ，求斐波那契数列的第 n 项。

假定从0开始，第0项为0。(n<=39)
### Demo
```
输入整数 n=5 

返回 5
```
### Analysis
最基础的做法就是用递归， 进阶做法用动态规划
### Code

```java
class Solution {
    public int Fibonacci(int n) {
         // 如果是 1 和 2 则直接返回
         if(n == 1 || n == 2){
             return 1;
         }
         // 否则的话动态规划
         n -= 2;
         
         int a = 1;
         int b = 1;
         int c = 0;
         while(n > 0){
            c = a + b;
            a = b;
            b = c;
            n--;
         }
         return c;
    }
}
```
