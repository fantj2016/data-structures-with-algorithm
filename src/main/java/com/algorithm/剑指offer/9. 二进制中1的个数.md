### Title
输入一个32位整数，输出该数二进制表示中1的个数。

**注意**

负数在计算机中用其绝对值的补码来表示。
### Demo
```
样例1
输入：9
输出：2
解释：9的二进制表示是1001，一共有2个1。

样例2
输入：-2
输出：31
解释：-2在计算机里会被表示成11111111111111111111111111111110，
      一共有31个1。
```
### Analysis
将n与n-1二进制取与(同时该位就变成0了), 看一共能取几次
### Code

```java
class Solution {
    public int NumberOf1(int n)
    {
        int count  = 0;
        while(n != 0){
            n = n&(n-1);
            count++;
        }
        return count;
    }
}
```
